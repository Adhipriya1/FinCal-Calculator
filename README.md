# 🚀 Features

## 1️⃣ SIP Calculator

Calculate the **future value of monthly SIP investments**.

**Inputs**

* Monthly Investment
* Expected Return Rate
* Investment Duration

**Outputs**

* Total Investment
* Estimated Returns
* Final Maturity Value

---

## 2️⃣ Lumpsum Calculator

Helps users estimate the **future value of a one-time investment**.

**Inputs**

* Initial Investment
* Expected Annual Return
* Time Duration

**Outputs**

* Total Returns
* Final Investment Value

---

## 3️⃣ Advanced Calculator

A flexible calculator allowing users to experiment with:

* Monthly investment
* Return rate
* Investment duration
* Investment growth scenarios

---

## 4️⃣ Goal-Based Investment Planner (I Know My Goal)

Users can set a **financial goal and timeline**.

**Inputs**

* Target Amount
* Time Period
* Expected Rate of Return

The system calculates:

* Required Monthly Investment
* Total Investment
* Expected Growth

Example
Goal: ₹10,00,000 in 10 years
The tool calculates **how much SIP you must invest monthly**.

# 🧮 Financial Formulas Used

## SIP Future Value Formula

```
FV = P × ((1 + r)^n − 1) / r × (1 + r)
```

Where

* **P** = Monthly Investment
* **r** = Monthly Interest Rate
* **n** = Number of Months
* **FV** = Future Value of Investment

---

## Lumpsum Future Value Formula

```
FV = P (1 + r)^n
```

Where

* **P** = Initial Investment
* **r** = Annual Interest Rate
* **n** = Number of Years
